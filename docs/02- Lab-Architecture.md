# Lab Architecture

## Overview

This lab consists of two virtual machines hosted in Oracle VirtualBox:

- Windows Server (Domain Controller)
- Windows 11 Client

The Windows Server virtual machine will host:

- Active Directory Domain Services (AD DS)
- DNS

The Windows 11 virtual machine will be joined to the Active Directory domain to simulate an enterprise workstation.
