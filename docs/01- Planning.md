# Project Planning
The purpose of this project is to design, build, and document a Windows Active Directory home lab to gain practical experience with enterprise identity and access management. The lab will provide a safe environment to practice Windows Server administration, Active Directory Domain Services (AD DS), DNS configuration, Group Policy, user and group administration, and troubleshooting.
## Project Goals
- Build a functional Windows domain environment.
- Install and configure Active Directory Domain Services.
- Configure DNS to support the domain.
- Create Organizational Units (OUs).
- Create and manage users and security groups.
- Join Windows client computers to the domain.
- Configure and test Group Policy Objects (GPOs).
- Document each implementation step and the reasoning behind configuration decisions.
- Practice troubleshooting common Active Directory issues.
