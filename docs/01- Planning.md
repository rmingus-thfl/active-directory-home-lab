# Project Planning

## Project Purpose

The purpose of this project is to design, build, and document an Active Directory home lab using Oracle VirtualBox. This lab will provide a hands-on environment to practice Windows Server administration, Active Directory Domain Services (AD DS), Domain Name System (DNS), user and group administration, Group Policy, and enterprise networking concepts.

## Project Objectives

- Build a Windows Active Directory domain environment.
- Install and configure Active Directory Domain Services (AD DS).
- Configure DNS to support the Active Directory domain.
- Create and organize Organizational Units (OUs).
- Create and manage user and group accounts.
- Join a Windows client computer to the domain.
- Configure and test Group Policy Objects (GPOs).
- Document each stage of the implementation.
- Practice troubleshooting and documenting solutions.

## Success Criteria

This project will be considered successful when:

- The Windows Server virtual machine is configured as a Domain Controller.
- Active Directory Domain Services and DNS are functioning correctly.
- A Windows 11 client successfully joins the domain.
- Users, groups, and Organizational Units (OUs) have been created and tested.
- Group Policy Objects (GPOs) have been configured and validated.
- The implementation is fully documented with screenshots and explanations.
